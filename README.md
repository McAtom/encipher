#Encipher - PHP代码加密工具#

本加密程序是用PHP代码写的，其中 ./lib/encipher.min.php 就是本加密程序加密后的代码。

加密后代码无需任何附加扩展，无需安装任何第三方组件，可运行于任何普通 PHP 环境下。

加密方法详见：./index.php

> 其中：./encoded目录下存放的就是加密后的实例。
